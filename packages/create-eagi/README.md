# create-eagi

Scaffolding bootstrapper for **EAGI (eagi-mcp-framework)** projects. 

EAGI is a unified, production-grade framework for building and orchestrating Model Context Protocol (MCP) servers.

## Quick Start

Initialize a new EAGI Domain Server project in seconds:

```bash
npm create eagi@latest my-new-domain
# or using pnpm
pnpm create eagi my-new-domain
# or using yarn
yarn create eagi my-new-domain
```

Follow the prompts to configure your domain and get up and running instantly.
